Python script to plot passage view of 2D airfoils in m'-theta plane

- Kiran Siddappaji
- GTSL, University of Cincinnati
- s2kn@mail.uc.edu

This is a python script originally created by Robert Knapke and modified by me.
Feel free to modify it for your own needs and share if it can be usable by others.

REQUIREMENTS:

1. Python 2.7
2. Numpy
3. Scipy
4. Matplotlib

Command: python passage_2dairfoilplot.py

Input: > 2d blade file from 3DBGB or any other code in the given format
       > filename and blade count in lines 19 and 24

Output: passage plot in m'-theta plane
